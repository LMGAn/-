# 内存池实现与性能优化项目

## 项目概述
基于C语言开发的高性能内存池管理系统，通过预分配机制和空闲链表技术解决传统动态内存管理的性能瓶颈问题，在特定场景下实现最高42.86%的性能提升。

## 核心技术亮点

### 🏗️ 架构设计
- **预分配机制**: 采用一次性大块内存分配策略，显著减少系统调用频率
- **空闲链表管理**: 实现O(1)时间复杂度的内存回收与重用
- **动态扩容策略**: 块数组按需翻倍扩展，平衡内存使用效率
- **线程安全保障**: 集成pthread_mutex实现多线程环境下的内存安全访问

### 🔧 关键算法实现
- **三级分配策略**: 空闲链表 → 当前块未使用空间 → 动态扩容新块
- **头插法空闲管理**: 高效的内存块回收机制
- **渐进式内存分配**: 避免一次性分配过多内存造成浪费

## 核心数据结构
```c
typedef struct {
    uint32_t elementSize;   // 元素大小
    uint32_t blockSize;     // 块容量
    uint32_t used;          // 当前使用量
    uint32_t block;         // 当前块索引
    uint32_t blocksUsed;    // 块数组容量
    uint8_t **blocks;       // 块指针数组
    poolFreed* freed;       // 空闲链表头
```
